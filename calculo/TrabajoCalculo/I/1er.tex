$\displaystyle \mathbf{ \int 
        x^{3} \sqrt{(4 - x^{2})^{2}} 
    \,dx}$ 

\begin{align*}
\int x^{3} \sqrt{(4 - x^{2})^{2}} &=\int x^{3}(4-x^{2})\,dx \\[6pt]
&= \int \big(4x^{3}-x^{5}\big)\,dx \\[6pt]
&= \int 4x^{3}\,dx - \int x^{5}\,dx \\[6pt]
&= 4\cdot\frac{x^{4}}{4} - \frac{x^{6}}{6} + C \\[6pt]
&= x^{4} - \frac{x^{6}}{6} + C
\end{align*}


\[
\boxed{\displaystyle 
    \int x^{3}(4 - x^{2})\,dx 
    = x^{4} - \frac{x^{6}}{6} + C
}
\]
