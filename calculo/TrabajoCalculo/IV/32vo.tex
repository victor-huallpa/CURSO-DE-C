$\displaystyle \mathbf{\int \frac{\sec^2(x)}{1+\tan x}\,dx}$
\nopagebreak
\[
\text{Sustituimos } t = \tan x \implies dt = \sec^2 x\, dx
\]
\[
\int \frac{\sec^2(x)}{1+\tan x}\,dx = \int \frac{dt}{1+t} = \ln|1+t| + C
\]
\[
\boxed{\displaystyle 
\int \frac{\sec^2(x)}{1+\tan x}\,dx = \ln|1+\tan x| + C
}
\]
