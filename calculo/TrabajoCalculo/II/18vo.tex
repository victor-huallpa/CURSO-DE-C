$\displaystyle \mathbf{\int \cos(x)\,e^{2\sin(x)}\,dx}$
\nopagebreak
\begin{align*}
\text{Sustituyendo: } &u = \sin(x) \;\;\Rightarrow\; du = \cos(x)\,dx. \\[6pt]
\int \cos(x)\,e^{2\sin(x)}\,dx
&= \int e^{2u}\,du \\[6pt]
&= \frac{e^{2u}}{2} + C \\[6pt]
&= \frac{e^{2\sin(x)}}{2} + C
\end{align*}
\[
\boxed{\displaystyle 
\int \cos(x)\,e^{2\sin(x)}\,dx = \frac{e^{2\sin(x)}}{2} + C.}
\]
