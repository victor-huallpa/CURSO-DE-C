$\displaystyle \mathbf{\int (e^{x} + 1)^{2} e^{x}\,dx}$
\nopagebreak
\begin{align*}
\text{Sustituyendo: } &u = e^{x} + 1 
\;\;\Rightarrow\; du = e^{x}\,dx. \\[6pt]
\int (e^{x} + 1)^{2} e^{x}\,dx 
&= \int u^{2}\,du \\[6pt]
&= \frac{u^{3}}{3} + C \\[6pt]
&= \frac{(e^{x} + 1)^{3}}{3} + C
\end{align*}
\[
\boxed{\displaystyle 
\int (e^{x} + 1)^{2} e^{x}\,dx 
= \frac{(e^{x} + 1)^{3}}{3} + C.}
\]
